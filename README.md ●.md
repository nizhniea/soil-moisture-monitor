# Soil Moisture Monitor

This is my first ever raspberry pi pico project

#Goal:
The goal is to make a Soil Moisture Monitor using the raspberry pi pico and a capacitive soil moisture sensor.
This repository includes both my practice programs (learning how to use the Pico) and the code used to build the final project.

#Ultimate Goal:
The oringinal idea behind this whole project was to save my bamboo plant that is not, putting it nicely, thriving currently. Using the soil moisture levels that is needed for bamboo, which would be accquired via google, I want to be able to compare the actual soil moisture level and keep it at the level it is supposed to be. That way, the plant will not be too dry or too wet. Eventually, I'd like to expand this into a system that I can monitor from my phone, making it possible to check the plant's soil moisture anytime.
